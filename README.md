# Mechanical Motion Lab：机械机构可视化学习平台

## 1. 项目背景

本项目是我为申请机械设计制造及其自动化第二学士学位所做的入门学习项目。

我本科阶段主修日语，机械基础相对薄弱，因此我选择从经典机械机构入手，使用 HTML、CSS 和 JavaScript 制作可视化学习页面，展示曲柄滑块机构和齿轮传动的基本运动原理，并整理中日英机械术语对照表。

## 2. 项目功能

- 曲柄滑块机构动画：展示旋转运动如何转化为往复直线运动
- 齿轮传动动画：展示齿轮啮合、反向旋转和传动关系
- 中日英机械术语表：整理基础机械词汇
- 学习反思：记录我对机械学习的初步理解和未来计划

## 4. 机械知识点

### 曲柄滑块机构

曲柄滑块机构可以将旋转运动转化为往复直线运动，常见于发动机活塞、冲压机、往复泵等机械系统中。

### 齿轮传动

齿轮传动通过轮齿啮合传递运动和动力。两个啮合齿轮旋转方向相反，齿轮半径或齿数不同会导致转速不同。
